Graphical Image as Password

About the Project

In this project, we introduce a new security primitive based on hard AI problems, namely, a novel family of graphical password systems integrating Captcha technology, which we call CaRP (Captcha as Graphical Passwords). CaRP is click-based graphical passwords, where a sequence of clicks on an image is used to derive a password. The notion of CaRP is simple but generic.

Technologies

  • Front End : HTML, Java, JSP
  • Scripts : JavaScript.
  • Server side Script : Java Server Pages.
  • Database : MySQL 5.0
  • Database Connectivity : JDBC.
